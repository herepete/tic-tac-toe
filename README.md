# tic-tac-toe
Tic-tac-toe

### Run game in P1 vs P2 mode
``./runner.py``

### Run game in P1 vs AI mode
``./runner.py ai``

### Enter a position
Should be in the format number space number
Examples:

1 1 is the top left square 

2 2 is the middle square

3 3 is the bottom right square

``` 
>> Welcome to TicTacToe! <<
[ ][ ][ ]
[ ][ ][ ]
[ ][ ][ ]

> X position: 2,2

 >> Welcome to TicTacToe! <<
[ ][ ][ ]
[ ][X][ ]
[ ][ ][ ]
```
